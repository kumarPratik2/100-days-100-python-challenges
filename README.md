# 100 Days, 100 Python Challenges

This repository contains my 100-day Python coding challenge where I solve one programming problem or build one small project every day.

The goal of this challenge is to improve my Python programming skills, problem-solving ability, consistency, and GitHub portfolio while pursuing MCA in AI/ML.

## Challenge Goals

- Practice Python daily
- Improve problem-solving skills
- Learn Data Structures and Algorithms step by step
- Build small Python projects
- Maintain consistent GitHub activity
- Document my learning journey
