# 100 Days, 100 Python Challenges 🐍

Welcome to my **100 Days of Python Challenge**!

I started this challenge to strengthen my Python fundamentals, improve problem-solving skills, build coding consistency, and document my learning journey publicly on GitHub.

Each day, I solve a programming problem or build a small Python project while gradually improving my coding practices, project structure, and documentation skills.

---

* Practice Python daily
* Improve problem-solving skills
* Learn Data Structures and Algorithms step by step
* Build small Python projects
