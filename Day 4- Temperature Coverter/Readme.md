# Temperature Converter 🌡️

A beginner Python project that converts temperatures between Celsius and Fahrenheit.

This project was created while learning Python fundamentals such as functions, user input handling, conditional statements, and type conversion.

---

## Features

* Celsius to Fahrenheit conversion
* Fahrenheit to Celsius conversion
* Function-based structure
* User input handling
* Temperature unit selection

---

## Concepts Used

* Functions
* Variables
* User input using `input()`
* Conditional statements (`if`)
* Type conversion using `float()`
* String methods like `.capitalize()`
* Basic mathematical formulas

---

## Conversion Formulas

### Fahrenheit to Celsius

```text id="tmpread1"
°C = 5 × (°F - 32) / 9
```

### Celsius to Fahrenheit

```text id="tmpread2"
°F = (°C × 9/5) + 32
```
