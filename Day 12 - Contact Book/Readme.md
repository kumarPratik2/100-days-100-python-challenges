# Contact Book 📒

A beginner Python project where the user can add, view, search, and delete contacts stored in a dictionary.

This project was created while learning core Python concepts such as functions, dictionaries, loops, conditions, and user input handling.

---

## Features

* Dictionary-based contact storage system
* Add new contacts with name and number
* View all saved contacts
* Search for a specific contact by name
* Delete a contact by name
* Input validation for invalid menu choices
* Continuation loop for repeated use

---

## Concepts Used

* Variables
* Dictionaries
* Functions
* `while` loops
* Conditional statements (`if`, `elif`, `else`)
* `.update()`, `.get()`, `.pop()` dictionary methods
* String methods (`.strip()`)
* f-strings
* Type conversion

---

## How to Run

1. Install Python
2. Download or clone this repository
3. Run the file using:

```bash
python "Contact Book.py"
```
