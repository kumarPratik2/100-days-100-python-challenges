# To-Do List CLI App 📝

A beginner Python project where users can create, view, and delete tasks using a simple command-line interface.

This project was created while learning core Python concepts such as functions, lists, loops, user input handling, and list operations.

---

## Features

* Add new tasks to the to-do list
* View all existing tasks
* Delete tasks from the list
* Menu-driven interface
* Task numbering using `enumerate()`
* Exit option to close the application

---

## Concepts Used

* Variables
* Functions
* Lists
* `while` loops
* `for` loops
* Conditional statements (`if`, `elif`, `else`)
* User input handling
* List methods (`append`, `pop`)
* String methods (`strip`, `title`)
* `enumerate()`

---

## How to Run

1. Install Python
2. Download or clone this repository
3. Run the file using:

```bash
python "To Do List.py"
```
