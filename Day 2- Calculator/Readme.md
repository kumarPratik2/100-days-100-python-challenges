# Simple Calculator 🧮

A beginner Python calculator project created while learning core Python concepts such as functions, loops, conditional statements, nested loops, and user input handling.

This calculator can perform basic arithmetic operations and allows the user to repeatedly perform calculations until they choose to exit.

---

## Features

* Addition
* Subtraction
* Multiplication
* Division
* Division-by-zero protection
* Repeated calculations using loops
* Function-based program structure
* Invalid operation handling
* Input validation for continuation (`y/n`)

---

## Concepts Used

* Variables
* Functions
* `while` loops
* Nested loops
* Conditional statements (`if`, `elif`, `else`)
* Boolean variables
* User input handling
* Type conversion using `float()`
* Basic program flow control

---

## How to Run

1. Install Python
2. Download or clone this repository
3. Run the file using:

```bash
python "Simple Calculator.py"
```
