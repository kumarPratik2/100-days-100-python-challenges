# BMI Calculator 🧮

A beginner Python project that calculates Body Mass Index (BMI) based on user input for height and weight.

This project was created as part of my **100 Days of Python Challenge**, focusing on strengthening Python fundamentals like loops, conditionals, exception handling, and user input validation.

---

## 🚀 Features

- Takes user input for height (meters) and weight (kg)
- Validates input using exception handling
- Prevents invalid height values (0 or negative)
- Calculates BMI using standard formula:
  BMI = weight / (height²)
- Classifies result into categories:
  - Underweight
  - Healthy
  - Overweight
  - Obese
- Allows repeated calculations using a loop
- Handles invalid user inputs gracefully
