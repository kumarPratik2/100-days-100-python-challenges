# Number Guessing Game 🎯

A beginner Python project where the player has 5 chances to guess a randomly generated number between 1 and 100.

This project was created while learning core Python concepts such as loops, conditions, exception handling, counters, and user input handling.

---

## Features

* Random number generation using Python's `random` module
* User input handling
* Exception handling for invalid inputs
* Guess counter system
* High/Low hints after every guess
* Limited attempts system
* Game Over condition

---

## Concepts Used

* Variables
* `while` loops
* Conditional statements (`if`, `elif`, `else`)
* Exception handling (`try-except`)
* Type conversion
* `break` and `continue`
* Counters

---

## How to Run

1. Install Python
2. Download or clone this repository
3. Run the file using:

```bash
python "Number Guessing Game.py"
```
